## Using this template

This repo is intended to be copied or forked as a starting point for portfolio projects.

- To create a new project, copy the files into a fresh repo and update `package.json` and `site.config.json`.
- Remove the `lib/cms/providers/mock.ts` or replace it with your real CMS provider when ready.

Reporting issues

This is a personal template — open an issue only if you think there's a bug in the starter code or a clear improvement everyone would benefit from.
